Unlock ID, try to make it unique
"parentUnlockId"
string
Setting this to another item's unlock ID, will require unlocking the other item first before making this mod available
You can reference an other workshop item's id for this. However it must then also include the prefix "modNode:".
As an example, to set the Badger-ex as the parent unlock, you must add: "parentUnlockId": "modNode:Badger-ex"

"node"
object
Vehicle object root. In the case of mechs, this would usually be the lower body.
Other vehicles generally don't have a split between upper/lower body, so this would just be the main body of the vehicle.
