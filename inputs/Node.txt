Mounts
"weaponMount"
object
Where a weapon is spawned for a given "weaponSlots" index.
Optional. Index count starts at 0.

object
Used for spawning equipped module objects. Similar to the weaponMount.
Optional. Index count starts at 0.
Only relevant for modules of the "Mounted" type, as other modules don't require a spawned object.

"destroyer"
object
Handles destruction of wall on collision.
Optional. Should also have collisionDamage set to function properly.
